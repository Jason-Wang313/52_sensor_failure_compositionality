# Sensor Failure Compositionality

Submission-hardening version: v2.

Decision: workshop-only.

The paper has a useful mechanism thesis: robot perception robustness should evaluate how sensor failures compose rather than reporting only isolated corruptions or dropout patterns. The v2 hardening pass narrows the claim. A pairwise interaction law is not universal: on a held-out triple-failure cell with a hidden common-cause term, pairwise MAE reaches 0.284 under a mild hidden triple and 0.758 under a strong hidden triple.

This repository should not claim deployment-ready robustness or a complete composition law. It is a workshop-level mechanism paper unless future evidence adds real robot data and higher-order/common-cause failure coverage.
